# NurseLink IPD — Frontend

Clinical Coordination System frontend for hospital staff & patient management.

## Stack
- Pure HTML/CSS/JS — no build step required
- Connects to NurseLink backend API on Render

## Deploy
This project is deployed on Vercel. Any push to `main` auto-deploys.
